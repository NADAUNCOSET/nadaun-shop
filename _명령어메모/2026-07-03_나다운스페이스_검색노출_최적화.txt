 blog: https://nadaunspace.blogspot.com  (소유 구글계정 = rbsent.info 계열)
 진단결과: 기술SEO 완벽(robots index·description·canonical·구조화데이터·사이트맵 다 정상).
          구글 ~10개 색인 O(단 7월 새글 미색인=지연) / 네이버 브랜드검색 노출 O.
          → 문제 = ①새 글 색인이 느림  ②경쟁키워드 순위 약함(블로그스팟 한계)
========================================================

■ A. 색인 속도 (새 글 바로 뜨게) — 최우선 ★★★
  [구글 서치콘솔]  https://search.google.com/search-console
   1) 나다운스페이스 소유 구글계정(rbsent.info)으로 로그인
   2) 속성 없으면 추가: URL 접두어 → https://nadaunspace.blogspot.com/
      (블로그스팟은 Blogger 소유자면 대부분 자동 소유확인)
   3) 좌측 'сайт맵(Sitemaps)' → 새 사이트맵 추가:
        sitemap.xml
        atom.xml?redirect=false&start-index=1&max-results=500   (블로그 글 색인용)
   4) 상단 'URL 검사' → 7월 새 글 주소 하나씩 입력 → "색인 생성 요청"
        예) /2026/07/capture-one-168.html, /2026/07/dji-4posmo-pocket-4p.html, /2026/07/3.html
      ※ 글 올릴 때마다 이 색인요청 습관화 = 몇 시간~하루면 구글 노출

  [네이버 서치어드바이저]  https://searchadvisor.naver.com
   1) 네이버 로그인 → 웹마스터도구 → 사이트 등록: https://nadaunspace.blogspot.com/
   2) 소유확인: HTML 태그 방식 → 메타태그 복사 → Blogger 테마 <head> 바로 밑에 붙여넣기
      (테마→HTML편집, 또는 알려주면 내가 넣어줌)
   3) 요청 → 사이트맵 제출: sitemap.xml
      요청 → RSS 제출: https://nadaunspace.blogspot.com/feeds/posts/default?alt=rss
   4) 요청 → '웹페이지 수집' → 새 글 URL 하나씩 수집요청
      ※ 네이버는 블로그스팟 자동수집이 느림 → 수집요청 꼭 하기

■ B. 순위(경쟁키워드 상위) — 중장기 ★★
   5) 제목·첫 문단·H2에 핵심키워드(장비명+렌탈/대여+지역) 배치 — 이미 잘 하심
   6) 백링크: 나다운 메인사이트/네이버블로그(issuehot_)/인스타에서 이 글로 링크
   7) 글 하단에 '관련글' 내부링크 3~4개 (체류시간·크롤링 ↑)
   8) 꾸준히 발행(주 2~3회) = 도메인 신뢰도 상승

■ C. 네이버 상위노출 현실
   - 블로그스팟은 네이버 통합검색에서 네이버블로그보다 밀림.
   - 렌탈 핵심 키워드는 네이버블로그(issuehot_/nadaunstudio 계열)도 병행 발행 추천.
   - 블로그스팟 = 구글 노출 주력 / 네이버블로그 = 네이버 노출 주력, 투트랙.
